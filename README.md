# php-basics-practice

## 概要

COACHTECH教材のハンズオン演習で作成した、シンプルな商品価格計算プログラムです。商品の価格、数量、消費税率を使って、合計金額を計算します。
ブランクからの復習として、変数と演算子の基礎を思い出しながら作成しました。

## 使用技術

- PHP : 8.5.1

## 学んだこと

約2ヶ月の休学期間を経ての再挑戦でしたが、こちらも初回学習時よりも構造の理解スピードが上がっていることを実感しました。

- **変数と文字列の使い分け**: クォーテーションやピリオドを使用し、変数と文字列をきれいに表示させました。
- **プログラムの実行順序（順次進行）の意識**: PHPはコードが上から順番に1行ずつ実行されるため、その計算の元となる単価・数量・消費税率の変数を正しい順序で定義・配置することを意識しました。
- **数字の見やすさ**: `number_format`を使用し、価格を読みやすくしました。

## 動作確認(閲覧方法)

ローカル環境（PC内）でページを確認する場合は、以下の手順で行えます。

1. **リポジトリのクローン**

```bash
git clone git@github.com:gomashio-no-omusubi/php-basics-practice.git
```

2. **クローンしたプロジェクトのディレクトリに移動します**

```bash
cd php-basics-practice
```

3. **ブラウザでの表示**

ブラウザで http://localhost:8000/php-basics-practice/price_calculator.php を開きます。
